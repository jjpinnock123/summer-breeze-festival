# Summer Breeze Reggae Festival website

## Open in VS Code
Open the `summer-breeze-site` folder in VS Code.

## Install dependencies
In the terminal run:

npm install

## Start the site

npm run dev

Then open http://localhost:3000

## Update the ticket link
Open `lib/siteData.ts` and replace:

ticketUrl: "#"

with your live Skiddle URL.
